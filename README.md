
# Соціальна Мережа (Social Network Project)

### Про проект
Це повноцінний веб-застосунок, розроблений як MVP (мінімально працююча версія) соціальної мережі. Проект поєднує в собі сучасний Backend на Django та адаптивний Frontend з використанням Bootstrap.

---

## Основний функціонал (MVP)
Проект реалізує ключову бізнес-логіку соціальної платформи:
Система автентифікації: Повна реєстрація нових користувачів та захищений вхід/вихід (Login/Logout).
Профілі користувачів: Персональні сторінки з інформацією про користувача та його активністю.
Керування публікаціями (CRUD): Можливість створювати, переглядати, редагувати та видаляти пости у стрічці.
Соціальна взаємодія: Реалізована основна логіка взаємодії (наприклад, коментарі або лайки).

---

## Технологічний стек
Проект побудований на базі наступних технологій:
Backend: Python 3.14, Django Web Framework (ORM, Templates, Auth).
HTML5, CSS3, JavaScript (ES6+), Bootstrap UI Framework.
Database: PostgreSQL або SQLite.
Version Control: Git (GitHub/GitLab).

---

## Структура команди
Розподіл ролей для ефективної розробки:
Backend Developer: Розробка серверної логіки, API та маршрутизації.

## Інструкція із запуску (Quick Start)
Дотримуйтесь цих кроків для локального запуску проекту:

1.  Клонування репозиторію:
    ```bash
    git clone <url_вашого_репозиторію>
    cd <назва_папки>
    ```
2.  Налаштування віртуального середовища:
    ```bash
    python -m venv venv
    source venv/bin/activate  # для Windows: venv\Scripts\activate
    ```
3.  Встановлення залежностей:
    ```bash
    pip install -r requirements.txt
    ```
4.  Застосування міграцій БД:
    ```bash
    python manage.py makemigrations
    python manage.py migrate
    ```
5.  Запуск сервера:
    ```bash
    python manage.py runserver
    ```
Проєкт буде доступний за адресою `http://127.0.0.1:8000/`
